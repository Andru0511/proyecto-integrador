Arbol Teoría de Grafos

Un árbol dirigido es un grafo dirigido que sería un árbol si no se consideraran las direcciones de las aristas. ... Los vértices de un árbol etiquetado de n vértices reciben normalmente las etiquetas {1,2, ..., n}. Un árbol regular u homogéneo es un árbol en el que cada vértice tiene el mismo grado.
